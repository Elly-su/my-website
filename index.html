About Me

I'm Elly, a Data Science student at the Cooperative University of Kenya with a strong passion for Python programming, machine learning, and building real-world tech solutions. I'm deeply curious about how data shapes decisions and the systems that power intelligent insights. My goal is to become a data engineer, using scalable tools and infrastructure to solve impactful problems. Outside of tech, I enjoy playing chess, where strategic thinking and pattern recognition fuel both my learning and creativity.

Over the past year, I've worked on several practical projects including a movie recommendation system using content-based filtering, a login authentication system using PHP and SQL, and an HR employee management database to strengthen my SQL and database design skills. My technical toolkit includes Python (with libraries like Pandas, NumPy, Scikit-learn, and Matplotlib), SQL, basic web development using HTML, CSS, and PHP, as well as version control with Git. I've also begun exploring Linux environments and recently started transitioning to Ubuntu to deepen my understanding of system-level workflows. I'm especially passionate about transforming raw data into structured, actionable insights and enjoy building pipelines that make this possible at scale. I plan to pursue a career in data engineering, where I can design systems that collect, process, and make data usable for analytics and AI. Outside the technical world, I'm a chess player who enjoys applying strategy and critical thinking both on and off the board.

My Portfolio

Project 1 Movie Recommendation System

Developed a content-based recommendation system using Python and Pandas to suggest movies based on user preferences. Leveraged cosine similarity and TF-IDF vectorization to match user inputs with relevant movie metadata. This project improved my understanding of natural language processing, feature engineering, and recommendation algorithms.

Project 2-HR Employee Management System

Built a CRUD-based employee management system using SQL for backend data handling. Designed relational tables to manage employee records including roles, departments, and leave balances. This project sharpened my database design skills, query optimization techniques, and understanding of data integrity and normalization principles.

Project 3-Simple Login Authentication Page

Developed a basic login page using HTML, CSS, PHP, and MySQL, hosted locally via XAMPP. Implemented secure user authentication with form validation, password hashing, and session management. This project gave me hands-on experience with backend development, database connectivity, and the foundations of web security.
